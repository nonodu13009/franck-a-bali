# Sons pour le site

## Son de gong balinais requis

Le site nécessite un fichier audio `bali-gong.mp3` pour le son de clic.

### Comment obtenir le fichier audio :

1. **Option 1 - Pixabay (Recommandé)**
   - Visitez : https://pixabay.com/sound-effects/search/balinese%20gong/
   - Cherchez "balinese gong" ou "gamelan"
   - Téléchargez un son court (1-2 secondes) de gong ou gamelan
   - Renommez le fichier en `bali-gong.mp3`
   - Placez-le dans ce dossier (`public/sounds/`)

2. **Option 2 - Freesound.org**
   - Visitez : https://freesound.org/
   - Cherchez "balinese gong" ou "gamelan bell"
   - Téléchargez un son libre de droits
   - Convertissez en MP3 si nécessaire
   - Renommez en `bali-gong.mp3`

3. **Option 3 - Fichier personnalisé**
   - Si vous avez votre propre enregistrement
   - Assurez-vous qu'il est court (< 2 secondes)
   - Format MP3, qualité optimisée pour le web
   - Nommez-le `bali-gong.mp3`

### Caractéristiques recommandées :
- **Format** : MP3
- **Durée** : 0.5 - 2 secondes
- **Qualité** : 128 kbps (équilibre taille/qualité)
- **Taille** : < 50 KB
- **Type** : Son de gong, cloche ou percussion balinaise

Une fois le fichier ajouté, le son jouera automatiquement à chaque clic sur le site.
